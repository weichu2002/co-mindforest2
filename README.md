## 🎯 项目介绍

MindForest 是一个基于阿里云ESA边缘计算平台的**ai思维树**应用，它将对话式AI与思维树可视化、多人实时协作相结合，为团队知识共创提供了全新的解决方案。
MindForest 首创 **"思维树型对话+边缘协同计算"** 双引擎架构，彻底改变了传统线性AI对话的局限性。我们不是在做另一个ChatGPT界面，而是构建一套**团队思维操作系统**。


### 🌳 思维树型对话：超越线性的思维革命

传统AI对话如同"单线程聊天"——一问一答，话题切换笨拙，历史回顾困难。MindForest采用**树状对话结构**，带来四大革命性优势：

#### 1️⃣ **非线性的思维探索**

传统：用户 → AI → 用户 → AI (线性链条)
MindForest：
用户问题
/
分支A解答 分支B深入
/ \ /
扩展A1 扩展A2 追问B1 示例B2
text


- **多路径并行**：一个核心问题可同时探索多个解答方向
- **分支独立发展**：每个分支保持完整上下文，互不干扰
- **随时回溯**：点击任意节点，立即回到当时的对话状态

#### 2️⃣ **可视化的思维图谱**
- **思维结构一目了然**：复杂讨论转化为清晰的树状图
- **决策路径可追溯**：每个结论的来源路径清晰可见
- **知识关联可视化**：不同话题间的联系直观展现

#### 3️⃣ **协作化的思维共创**
- **主持人建立主干**：确定讨论框架和核心问题
- **协作者发展分支**：团队并行探索不同解决方案
- **分支状态同步**：实时查看队友的思维进展和成果

#### 4️⃣ **结构化的知识沉淀**
- **对话即知识库**：每次讨论自动形成结构化知识树
- **可复用模板**：成功的讨论模式可保存为思维模板
- **渐进式完善**：知识树随时间生长、优化、丰富

### ⚡ 边缘计算加速：毫秒级的全球协作体验

传统协作工具受限于中心服务器延迟，全球团队体验割裂。MindForest基于**阿里云ESA边缘计算平台**，实现：

####
**全球毫秒级响应**
**智能地域路由**：用户自动连接最近的边缘节点
**本地化AI计算**：AI请求就近处理，无需绕道中心
**实时状态同步**：团队状态通过EdgeKV全球秒级同步



### 🌟 核心特性

**🤝 分叉式协作模式**
- **主持人模式**：创建房间，建立初始思维树
- **协作者模式**：加入房间，自动复制思维树，独立分叉发展（可以查看历史对话记录）
- **分支状态同步**：实时显示其他成员的思维树状态（节点数、最后更新时间）

**🌐 ESA边缘加速**
- 全球多节点部署：北京、新加坡、法兰克福、硅谷
- 边缘计算加速：AI推理请求通过最近节点处理
- KV存储协作：使用ESA EdgeKV实现实时状态同步

**🌳 思维树可视化**
- 交互式思维图谱：Cytoscape.js构建的可视化树状结构
- 多分支显示：同时查看不同协作者的思维分支
- 节点回溯：点击任意节点跳转到对应对话位置

**💬 智能对话引擎**
- 支持DeepSeek
- Markdown + LaTeX 完整渲染
- Word文档上传解析功能
- 可在每个分叉中自定义AI人设（System Prompt）
